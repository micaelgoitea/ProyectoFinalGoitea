El proyecto consiste en el diseño de una app web que permite al personal de un restaurante tomar los pedidos de cada una de las mesas
y generar la cuenta de dicha mesa en tiempo real de una manera super práctica y eficaz.
La aplicación divide por secciones los platos especiales, como los Platos aptos para celíacos y los Vegetarianos, para que el mozo
acceda rápidamente a estas opciones y pueda detallarle al cliente la disponibilidad de la carta apto para ellos.
Una vez que los pedidos finalizaron, se puede hacer el cierre de cuenta para comenzar una nueva e incluso borrar solamente un plato del pedido
que haya sido cargado por error.
